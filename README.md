# Prompt Engineering Internship – Future Interns

This repository contains the tasks completed during the **Prompt Engineering Internship (2026)** organized by Future Interns.

The internship focuses on using **AI tools and prompt engineering techniques** to create practical solutions for businesses such as website content generation, marketing ad scripts, and SEO blog strategies.

---

# Tools Used

- ChatGPT (AI Prompt Generation)
- GitHub (Project Documentation & Submission)
- Basic SEO Research Tools

---

# Tasks Overview

## Task 1 – AI Website Copy Generator for Local Businesses

This task demonstrates how AI prompts can generate website copy for local businesses such as salons, cafes, clinics, and agencies.

### Key Features
- Homepage headline and introduction
- Services page content
- Call-to-action sections
- Business-focused website content

### Example Business
Brew Haven Cafe – Bangalore

The generated content includes:
- Homepage copy
- Service descriptions
- Customer-focused call-to-action

---

## Task 2 – AI UGC Ad Script Generator

This task focuses on generating **UGC-style ad scripts** using AI.

UGC ads are widely used on platforms like Instagram Reels, Shorts, and paid social media ads.

### Key Features
- Multiple ad hooks
- UGC-style storytelling
- Problem → Solution → CTA framework
- Conversion-focused ad scripts

### Example Product
GlowSkin Vitamin C Face Serum

The generated content includes:
- Multiple ad hooks
- UGC ad script
- Call-to-action
- Optional social media caption

---

## Task 3 – AI SEO Blog & Content Cluster Generator

This task demonstrates how AI can generate **SEO-focused blog strategies and content clusters** for business websites.

### Key Features
- SEO pillar blog
- Supporting blog articles
- Keyword-focused structure
- Local SEO optimization
- Internal linking strategy

### Example Business
SmileCare Dental Clinic – Bangalore

The generated content includes:
- Pillar blog
- 4 supporting blog articles
- SEO keyword strategy
- Internal linking suggestions
